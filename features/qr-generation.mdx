## How It Works

QRcodeer generates QR codes in real-time as you type. Simply enter any text or URL in the content field, and your QR code appears instantly in the preview area.

### Supported Content Types

You can create QR codes from:

- **URLs** - Website links, social media profiles, online forms
- **Plain Text** - Contact information, messages, Wi-Fi credentials
- **Email Addresses** - Quick mailto links
- **Phone Numbers** - Tap-to-call functionality
- **Any Text Content** - The QR code library supports any UTF-8 text

<Info>
QR codes can store up to several thousand characters, but shorter content produces simpler, more scannable codes.
</Info>

## Error Correction Levels

QRcodeer automatically adjusts the error correction level based on your customization:

<Tabs>
  <Tab title="Standard (Level M)">
    Used for basic QR codes without a center image.
    
    - **Recovery Rate**: ~15% of data can be restored if damaged
    - **Best For**: Simple URLs and text
    - **Code Density**: Moderate
  </Tab>
  <Tab title="High (Level H)">
    Automatically activated when you add a center image/logo.
    
    - **Recovery Rate**: ~30% of data can be restored if damaged
    - **Best For**: QR codes with branding or logos
    - **Code Density**: Higher (more modules)
    
    <Note>
    Level H error correction ensures your QR code remains scannable even with a logo covering the center.
    </Note>
  </Tab>
</Tabs>

## Technical Details

QRcodeer uses the `qrcode.react` library to generate high-quality SVG-based QR codes that are:

- **Scalable** - Vector graphics remain crisp at any size
- **Fast** - Instant generation with no server requests
- **Accurate** - Industry-standard QR code encoding
- **Accessible** - Properly labeled for screen readers
